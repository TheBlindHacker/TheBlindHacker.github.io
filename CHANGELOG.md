# Changelog

All notable changes to `TheBlindHacker.github.io` will be documented in this file.

## [Unreleased] - 2026-02-11

### Added
- **Typography**: Integrated `OpenDyslexic` and `Lexend` fonts for enhanced readability.
- **Theme**: Implemented high-contrast `Dark Theme` (#121212 background, #E0E0E0 text) with Neon Blue/Green accents.
- **Layout**: "Mobile-First" responsive design, sticky navigation, and media grid system.
- **Accessibility**: Added "Skip to Content" link, ARIA landmarks, and focus states for keyboard navigation.
- **Content**:
    - Expanded `profiles.md` with verified social media links, conference appearances (Hack Space/Red Con), and podcasts.
    - Updated `hackingwhileblind.md` with a "Terminal Survival Guide" focusing on CLI accessibility (redirection, grep) and tool usage.

### Changed
- **Refactoring**: Cleaned up SCSS files, removing unused styles (`light.scss`, `grey.scss`).
- **Organization**: Categorized media appearances into Conferences, Podcasts, and Interviews.
